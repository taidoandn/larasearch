# Responsive Rules

## Breakpoints

- Desktop: >= 1280px
- Tablet: 768px to 1279px
- Mobile: < 768px

## Search Results Page

### Desktop

- Full-width list by default
- When a job is selected:
    - list width ~48%
    - summary detail panel width ~52%

### Tablet

- Full-width list by default
- Selected job opens a right slide-over summary panel
- Top filter/search bar remains usable and compact

### Mobile

- Results list remains full width
- Selecting a job navigates to a full-screen detail page
- Filters open in bottom sheet or full-screen overlay
- Active filters appear as horizontal chips

## Full Job Detail Page

### Desktop

- Main content prioritized
- Sidebar visually secondary

### Tablet

- Sidebar may stack below or reduce width

### Mobile

- Single-column reading layout
- Sticky top action bar allowed if needed

## Benchmark Panel

### Desktop

- Overlay or anchored structured panel

### Tablet

- Full overlay or sheet

### Mobile

- Full-screen layer
